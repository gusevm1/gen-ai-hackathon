## Decisions Made
- **CheerioCrawler over PlaywrightCrawler:** Homegate uses Vue.js SSR, embedding all data in __INITIAL_STATE__ script tags. No browser rendering needed, 10-50x faster and much cheaper on Apify credits.
- **Search-page-only extraction:** Search results contain enough fields (url, title, price, rooms, address, livingSpace, coordinates) to satisfy PropertyListingSchema required fields. Detail pages can be added later.
- **Crawler ref pattern for pagination:** Used a shared `crawlerRef` object so the request handler can call `crawler.addRequests()` for pagination without needing a router.
- **No specific proxy group:** Let Apify decide the optimal proxy for cost efficiency. Can be upgraded to residential if blocking occurs.
- **Field name alignment:** transformListing() outputs field names matching normalizeHomegate() access patterns (url, price, rooms, livingSpace, address with raw/street/zip/city, latitude, longitude, description, images, propertyType, agencyName, currency).
