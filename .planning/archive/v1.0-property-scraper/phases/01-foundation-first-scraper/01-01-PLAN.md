**Step 2 — Create src/schema/listing.ts:**

Define the PropertyListing Zod schema exactly matching user decisions:

```typescript
// Required fields (per locked user decisions):
// - url (valid URL), title (non-empty), listingType ('rent' | 'buy')
// - price (positive number), rooms (positive number)
// - address: { raw (string), street? (string), zip? (string), city (string, required), canton (string, required) }
// - source (string, e.g. "homegate"), scrapedAt (ISO datetime string)

// Optional comprehensive fields:
// - livingSpace (number), floor (number), yearBuilt (number)
// - description (string — full text, no truncation per user decision)
// - imageUrls (array of URL strings — URLs only, never files per user decision)
// - features (array of strings), propertyType (string)
// - agencyName (string), agencyContact (string)
// - latitude (number), longitude (number)
// - currency (string, default 'CHF')
```

Export: `PropertyListingSchema`, `PropertyListing` (inferred type), `AddressSchema`.

Use Zod v4 API (`import { z } from "zod"`). Use `z.object()` (not `z.strictObject()`). Use `.url()` on z.string() for URL validation (Zod v4 still supports this on string).

**Step 3 — Create src/scrapers/types.ts:**

```typescript
export interface ScrapeOptions {
  dryRun: boolean;
}

export type RawRecord = Record<string, unknown>;

export interface ScraperAdapter {
  readonly name: string;
  scrape(options: ScrapeOptions): Promise<RawRecord[]>;
}
```

This is the adapter interface that HomegateAdapter (Plan 02) and FlatFoxAdapter (Phase 2) will implement.

**Step 4 — Create src/output/jsonl-writer.ts:**

Export `writeJsonl(records: PropertyListing[], outputDir: string): Promise<string>`:
- Create `outputDir` recursively with `fs.promises.mkdir(dir, { recursive: true })`
- Open write stream to `{outputDir}/listings.jsonl`
- Write each record as `JSON.stringify(record) + '\n'`
- Return a Promise that resolves with the file path when the stream finishes
- Handle stream errors by rejecting the Promise

Import `PropertyListing` type from `../schema/listing.js`.
