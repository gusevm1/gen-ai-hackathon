         - Catches httpx.HTTPStatusError: if 404 -> raise HTTPException(404, "Listing {pk} not found on Flatfox"), else -> raise HTTPException(502, "Flatfox API error")

         - Returns the FlatfoxListing model
       - Import httpx for exception types

    3. Update `backend/app/main.py`:
       - Add lifespan context manager for httpx client cleanup (Pattern 6 from RESEARCH.md):
         ```python
         from contextlib import asynccontextmanager
         from app.services.flatfox import flatfox_client
         from app.routers import listings

         @asynccontextmanager
         async def lifespan(app: FastAPI):
             yield
             await flatfox_client.close()

         app = FastAPI(title="HomeMatch API", version="0.2.0", lifespan=lifespan)
         ```
       - Include listings router: `app.include_router(listings.router)`
       - Keep existing health check endpoint and CORS middleware
