  <files>extension/src/entrypoints/content/App.tsx, extension/src/entrypoints/content/components/ScoreBadge.tsx</files>
  <action>
1. Update `extension/src/entrypoints/content/App.tsx`:

   a. Add imports:
      - `import { activeProfileStorage } from '@/storage/active-profile'`

   b. Add state for stale tracking:
      - `const [isStale, setIsStale] = useState(false)`
      - `const isStaleRef = useRef(false)` (for shadow root re-renders)
      - `const scoredProfileIdRef = useRef<string | null>(null)` -- tracks which profile the current scores belong to

   c. Add `useEffect` for profile change watching:
      ```
      useEffect(() => {
        const unwatch = activeProfileStorage.watch((newProfile, oldProfile) => {
          if (oldProfile && newProfile && oldProfile.id !== newProfile.id) {
            // Profile changed -- mark existing badges as stale
            isStaleRef.current = true;
            setIsStale(true);
            rerenderAllBadges(openPanelRef.current);
          }
        });
        ctx.onInvalidated(() => unwatch());
        return () => unwatch();
      }, [ctx, rerenderAllBadges]);
      ```

   d. On initial mount, load the current active profile into `scoredProfileIdRef`:
      ```
      useEffect(() => {
        activeProfileStorage.getValue().then(profile => {
          scoredProfileIdRef.current = profile?.id ?? null;
        });
      }, []);
      ```

   e. Modify `handleScore` to:
      - Before scoring, read `activeProfileStorage.getValue()` and store the profile ID in `scoredProfileIdRef.current`
      - After scoring completes successfully, clear stale state: `isStaleRef.current = false; setIsStale(false);`
      - Re-render all badges after clearing stale (they already get re-rendered by the progressive callback, but add a final `rerenderAllBadges(openPanelRef.current)` after the scoring loop to ensure stale flag is cleared visually)

   f. Modify `renderBadge` to pass `isStale={isStaleRef.current}` to `ScoreBadge` and `SummaryPanel`:
      ```
      mount.root.render(
        <div className="inline-block">
          <ScoreBadge
            score={score}
            listingId={pk}
            isPanelOpen={panelOpenId === pk}
            isStale={isStaleRef.current}
          />
          <SummaryPanel score={score} listingId={pk} isOpen={panelOpenId === pk} isStale={isStaleRef.current} />
        </div>,
      );
      ```

2. Update `extension/src/entrypoints/content/components/ScoreBadge.tsx`:

   - Add `relative` to the button className (needed for absolute positioning of stale indicator)

     - Add `opacity-60 ring-2 ring-amber-400/70` to the button
     - Remove `hover:shadow-lg` (stale badges shouldn't look interactive in the same way)
