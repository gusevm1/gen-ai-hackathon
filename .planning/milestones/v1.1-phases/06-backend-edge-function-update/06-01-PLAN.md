
       ```typescript
       const { data: profile, error: profileError } = await supabase
         .from("profiles")
         .select("id, preferences")
         .eq("is_default", true)
         .single();
       ```
       - Only select `id` and `preferences` (NOT `select('*')` -- avoid leaking unnecessary data).
       - `.single()` returns exactly one row or errors.

       Previously: `{ ...body, user_id: data.user.id }`
