Change to:

```typescript
if (cached && !cached.stale) {
    // DB-02: Reject stale v1 cache entries -- require schema_version >= 2
    const schemaVersion = cached.breakdown?.schema_version ?? 0;
    if (schemaVersion >= 2) {
        return new Response(
            JSON.stringify(cached.breakdown),
            {
                status: 200,

        );
    }
    // v1 entry found but schema too old -- fall through to backend for re-scoring
}
```

The `?? 0` handles both missing `schema_version` (v1 entries) and null values. If `schema_version < 2`, we fall through to the backend proxy call which will re-score using the new hybrid pipeline.
