**State:**
```typescript
type ConversationPhase = 'idle' | 'naming' | 'chatting';

interface Message {
  id: string;

  content: string;
  timestamp: Date;
}
```

**Mock AI response function** (to be replaced in Phase 15):
```typescript
const mockAIResponse = async (userMessage: string): Promise<string> => {
  return new Promise((resolve) => {
    setTimeout(() => {
      resolve("That sounds great! Could you tell me more about your budget range and preferred neighborhood? Also, are there any specific amenities nearby that are important to you — like public transport, schools, or parks?");
    }, 1500);
  });
};
```

**Handlers:**
- `handleStartCreating(description: string)`: Set `pendingDescription` to description, set phase to `'naming'`
- `handleNameSubmit(name: string)`: Set `profileName` to name, set phase to `'chatting'`, create first user message from `pendingDescription` (use `crypto.randomUUID()` for id), add to messages, then trigger mock AI response
- `handleSendMessage(content: string)`: Create user message, add to messages, set `isTyping` true, call `mockAIResponse`, then create assistant message from response, add to messages, set `isTyping` false
- Auto-scroll: `useEffect` that calls `bottomRef.current?.scrollIntoView({ behavior: 'smooth' })` when `messages` or `isTyping` changes

**Layout JSX:**
The outer container must account for the dashboard layout. The `<main>` in layout.tsx has `flex-1 p-4`. The chat page needs to fill this space. Use:
```tsx
<div className={cn(
  "flex flex-col mx-auto w-full max-w-3xl",
  "h-[calc(100dvh-3.5rem-2rem)]",  // 100dvh minus header (3.5rem/56px) minus main padding (2rem/32px top+bottom)
  phase === 'idle' ? "justify-center" : "justify-end"
)}>
```

**Conditional rendering:**
- When `phase === 'idle'`: Show `<ChatInput onSend={handleStartCreating} showStartButton={true} />`
- When `phase === 'naming'`: Show `<ProfileNamePrompt onSubmit={handleNameSubmit} />` centered
- When `phase === 'chatting'`:
  - Show message thread: `<div className="flex-1 overflow-y-auto space-y-6 pb-4">` containing mapped messages
  - Each message renders as:
    - Assistant: `<div className="flex gap-3 rounded-lg bg-muted/50 p-4">` with `<AIAvatar />` on the left, content + timestamp on the right
    - User: `<div className="flex justify-end"><div className="max-w-[80%] p-4">` content right-aligned + timestamp
  - Timestamps: `<span className="text-xs text-muted-foreground">` showing `timestamp.toLocaleTimeString()`
  - If `isTyping`: show `<div className="flex gap-3 rounded-lg bg-muted/50 p-4"><AIAvatar /><TypingIndicator /></div>`
  - Bottom scroll anchor: `<div ref={bottomRef} />`
  - Below thread: `<ChatInput onSend={handleSendMessage} disabled={isTyping} showStartButton={false} />`

Also show profile name as a subtle header when in chatting phase:
