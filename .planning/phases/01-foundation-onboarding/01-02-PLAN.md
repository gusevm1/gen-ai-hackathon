    **Layout -- two-panel design:**

    **Left panel -- Category Prompts with Curated Suggestions:**
    Display 5-6 category cards, each with a heading and curated suggestion chips the user can click to add. Categories and suggestions (Claude's discretion per CONTEXT.md):

    1. "Surroundings & Location" -- suggestions: Near public transport, Lake/river view, Quiet neighborhood, Near schools, Near shops/restaurants, Near park/green space, Near city center
    2. "Building & Quality" -- suggestions: Modern renovation, Well-maintained building, Energy efficient, Good insulation, New construction, Historic charm
    3. "Outdoor Space" -- suggestions: Large balcony, Private garden, Rooftop terrace, South-facing, Sunny exposure
    4. "Lifestyle & Comfort" -- suggestions: Open floor plan, High ceilings, Bright/lots of light, Separate kitchen, Walk-in closet, Home office space
    5. "Practical" -- suggestions: Low utility costs, Low tax municipality, Good internet connectivity, Short commute, Ample storage
    6. "Custom" -- free-text Textarea where user types anything, with an "Add" button

    Each suggestion chip is a Badge component. Clicking it adds a SoftCriterion object `{ id: uuid(), category: categoryName, text: suggestionText, isCustom: false }`. Clicking again removes it. Custom text adds `{ id: uuid(), category: 'Custom', text: userText, isCustom: true }`.

    **Right panel -- Selected Criteria Summary:**
    A scrollable list (ScrollArea) showing all selected criteria grouped by category. Each item has a remove (X) button. Shows count: "N criteria selected". Empty state: "No criteria selected yet -- click suggestions or type your own."

    **Bottom -- LLM Chat Placeholder:**
    Create `SoftCriteriaChat.tsx`: A compact chat-like UI with an Input field and send Button. In Phase 1 this is LOCAL ONLY (no backend).     - If input contains "school" -> suggest "Near schools"
    - If input contains "quiet" -> suggest "Quiet neighborhood"
    - If input contains "transport" or "train" or "bus" -> suggest "Near public transport"
    - If input contains "garden" -> suggest "Private garden"
    - Otherwise: add the full text as a custom criterion

    Display a note: "AI-powered refinement coming soon -- for now, try describing what matters to you!" This gets replaced with real Claude integration when backend exists in Phase 2.
